# [alghijudge-api](https://github.com/darklordace/alghijudge-api)

## Updating Test Cases
1. Clone this repo
2. Add test cases accordingly in `/static/` folder
3. Update `numberOfCasesDict` in `/controllers/submitcode.js`
4. Commit, and submit pull request
